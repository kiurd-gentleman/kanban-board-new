## How to run this project

To run this project you need PHP8 and node 16up. Because I use laravel 9 and vite engine. I also use composition API

- run - `composer install`
- run - `npm install`

after that just configure the env and database connection

- run - `copy .env.example .env`
- run - `php artisan key:generate`
- run - `php artisan serve`
- run - `npm run dev`
